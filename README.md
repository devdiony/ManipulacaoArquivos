Atividade Prática: Manipulação de Arquivos, Exceções e Refatoração POO

Este projeto implementa as melhorias e extensões propostas na atividade "Arquivos de Texto e Exceções em Java". O objetivo foi pegar um código-fonte base funcional, porém frágil, e aplicar os princípios de POO e tratamento de exceções para torná-lo mais robusto, seguro e organizado.
